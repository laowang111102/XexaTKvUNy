# 项目概述 python582

本项目是一个基于 Python 的课程成绩管理系统，采用 Django 框架进行开发，使用 MySQL 数据库进行数据存储，兼容 Python 3.7 以上的版本，开发环境为 PyCharm。

# 技术栈

- 后端：Python
- 框架：Django
- 数据库：MySQL
- 开发工具：PyCharm
- Python 版本：3.7 及以上

# 功能模块

系统主要包括以下模块：

- 学生管理
- 成绩管理
- 信息管理
- 教师管理
- 设置

# 系统角色

系统包含三个角色：管理员、学生、教师。

## 管理员

- 修改教师相关信息
- 修改学生相关信息
- 添加教师相关信息

## 学生

- 查看个人信息
- 查看个人成绩
- 修改个人密码

## 教师

- 查看学生信息
- 修改学生信息
- 添加学生信息
- 查看学生成绩
- 修改学生成绩
- 添加学生成绩
- 修改个人密码

# 运行环境

- Python 3.7 及以上版本
- Django 框架
- MySQL 数据库
